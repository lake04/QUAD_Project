<h2>📖 Project Overview</h2>
<p align="justify">
  <b>테바누이</b>는 멸망해가는 정령의 섬을 배경으로 한 메트로배니아 게임입니다.<br/>
  정령과의 공명을 통해 지상과 수중을 넘나드는 탐험을 제공하며, 메트로배니아 특유의 긴장감 넘치는 전투와 레벨 디자인을 특징으로 합니다.
</p>
